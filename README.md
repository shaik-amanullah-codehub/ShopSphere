# ShopSphere – Omnichannel Retail Inventory & Customer Engagement System
 
## 🏬 Overview
ShopSphere is an Omnichannel Retail Inventory and Customer Engagement System built to support
modern retail chains and e-commerce platforms. The system enables seamless inventory tracking,
order processing, customer loyalty management, analytics, and marketing campaigns.
 
This repository contains the **React JS Frontend (Phase 1)** using **dummy JSON data**, designed to
be seamlessly integrated with **Spring Boot REST APIs** in Phase 2.
 
---
 
## 🧠 Key Capabilities
- Multi-channel retail experience
- Customer & Admin portals
- Inventory & order lifecycle management
- Loyalty & engagement features
- Analytics-driven decision making
- Backend-ready frontend architecture
 
---
 
## 🧑‍🤝‍🧑 Team & Responsibility Split
 
### Customer Portal
- Product browsing & search
- Cart & checkout flow
- Order history & loyalty dashboard
 
### Admin Portal
- Inventory & product management
- Order processing & fulfillment
- Sales analytics & campaign tracking
